READ THIS BEFORE PLAYING PLEASE!

This game is a personal project of mine and doesn't profit off any creators. I am not intending to commit copyright infringement. All cartoon characters belong to their rightful owners. This game uses Wikipedia's api to select characters and estimate their stats, and the game scrapes images from Bing to get the character's image and background, meaning this game is likely to be inaccurate and make mistakes, especially when it comes to choosing a background. Those mistakes include but are not limited to: choosing the wrong image for a character, giving the character abilities it shouldn't have and vice versa, or giving both characters the same image. Mistakes like these are out of my control, so only report bugs involving the game itself, not the api. I recommend you play the game in fullscreen (F11 in Windows, Command + Control + F or Fn + F on Mac) since the game runs on html, not a canvas. Press enter to begin playing.

Thanks for playing my game
